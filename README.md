\# Social Media Agent 



This is a Streamlit-based AI app that generates social media captions using a locally hosted language model.



\# Features

\- Choose platform (Instagram, Twitter, LinkedIn)

\- Select tone (Casual, Professional, Funny, Inspirational)

\- Enter your brand/topic

\- Choose number of days of content

\- Generate and download captions



\# Model Used

\- "distilgpt2" from Hugging Face Transformers

\- Runs fully offline using "transformers" and "torch"



\# How to Run

1\. Create a virtual environment (optional but recommended)

2\. Install dependencies: streamlit, transformers and torch

3.Run the app
